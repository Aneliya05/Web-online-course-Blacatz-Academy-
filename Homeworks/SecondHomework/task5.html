HTML
Hyper Text Markup Language. Програмен език, който се използва за разработката на уеб сайтове, като служи оформя структурата на сайта.

CSS
Cascading Style Sheets. Компютърен език, използващ се за стилизиране на уеб сайтове.

JavaScript
Програмен език, допринасящ за динамичността на уеб сайтовете чрез писане на функции, които също така улесняват потребителското преживяване.
